## 工作流程

### 第一步：分析領據結構

1. 用 `pandoc` 提取文字內容，快速掌握欄位配置
2. 識別需要填寫的欄位（姓名、身分證、地址、銀行等）
3. 確認表格中的圖片黏貼區位置

### 第二步：使用 Python 腳本填寫

執行 `scripts/fill_receipt.py` 來自動填寫：

```bash
python scripts/fill_receipt.py <input.docx> [input2.docx ...] [--output-dir <dir>]
```

腳本會：
- 自動辨識並填寫所有文字欄位
- 在姓名欄插入簽名圖片
- 在「黏貼處」插入身分證影本
- 在匯款資料區插入存摺影本
- 輸出為 `原檔名_已填寫.docx`（不覆蓋原檔）

### 第三步：驗證

填寫完成後用 `pandoc` 確認文字欄位是否正確填入。提醒使用者用 Word 開啟確認圖片位置。

## 欄位辨識規則

領據中的欄位通常以標籤+冒號的形式出現，辨識邏輯如下：

| 欄位 | 辨識方式 | 填入位置 |
|------|----------|----------|
| 姓名 | 段落含「姓名：」和「簽名或蓋章」 | 「姓名：」後方，加上簽名圖 |
| 身分證字號 | 段落含「身分證字號：」 | 冒號後方 |
| 戶籍地址 | 段落含「戶籍地址」和「含里」 | 冒號後方 |
| 服務單位 | 段落含「服務」和「連絡電話」 | 「單位：」後的空白 run |
| 連絡電話 | 同上 | 「連絡電話：」後方 |
| 銀行匯款 | 段落含「銀行」「郵局」「分行」 | 各對應欄位後方 |

## 圖片插入規則

| 圖片 | 目標位置 | 尺寸策略 |
|------|----------|----------|
| 簽名檔 | 姓名 run 後方 | 寬度 2.5cm（橫式小圖） |
| 身分證影本 | 表格中「黏貼處」儲存格 | 寬度 8.5cm（配合儲存格寬度） |
| 存摺影本 | 「匯款資料」下方空白列 | 依圖片方向自動調整（直式用高度、橫式用寬度） |

圖片尺寸應根據實際圖片的長寬比自動計算，避免變形。

## 技術注意事項

### python-docx 合併儲存格問題

Word 表格的合併儲存格在 python-docx 中會重複出現。遍歷時必須用 `id(cell)` 去重：

```python
seen_cells = set()
for cell in row.cells:
    if id(cell) in seen_cells:
        continue
    seen_cells.add(id(cell))
```

### 段落去重

同理，遍歷所有段落時用 `id(para)` 追蹤已處理的段落，並用 `filled` 集合確保每個欄位只填寫一次。

### 簽名圖片定位

用 `para.add_run()` 新增圖片 run 後，透過 lxml 的 `addnext()` 將圖片 run 移到姓名文字 run 的正後方，確保簽名出現在名字旁邊而非段落末尾。

## 支援的領據類型

- 校外講師鐘點費領據
- 外聘講座費領據
- 出席費/審查費領據
- 其他具有類似欄位結構的教育機構付款表單

不同學校的領據格式可能略有差異，腳本的欄位辨識是基於關鍵字匹配，對大多數標準格式都能適用。如遇到無法自動辨識的欄位，會跳過並提醒使用者手動填寫。

## 常見問題

**Q：地址要用身分證上的還是現居地址？**
A：領據一般填戶籍地址（含里、鄰），使用身分證背面上的地址。目前預設填入身分證上的地址。

**Q：「已有資料留存本校」要勾哪個？**
A：如果之前已經提供過匯款資料給該校，勾「是」。否則勾「否」並填寫完整銀行資訊。腳本預設不更改此勾選狀態。

**Q：可以只填文字不貼圖片嗎？**
A：可以。如果圖片檔案不存在，腳本會跳過圖片插入，只填寫文字欄位。
