# 筆記品質規範

## 通過標準

### 國際論文筆記（paper-note-template）

| 指標 | 最低要求 |
|------|---------|
| 總行數 | >= 120 行 |
| LaTeX 公式 | >= 2 處（若論文有公式） |
| 圖片/圖表 | >= 1 張（若可取得） |
| 必含章節 | 元資訊、一句話總結、核心貢獻、方法詳解、實驗、批判性思考 |
| wikilink | >= 3 個概念連結 |
| 禁止事項 | 不得有空白章節、不得複製貼上摘要充數 |

### 台灣博碩士論文筆記（thesis-note-template）

| 指標 | 最低要求 |
|------|---------|
| 總行數 | >= 100 行 |
| 必含欄位 | 學校、系所、指導教授、學位、民國年/西元年 |
| 必含章節 | 元資訊、摘要（中英）、研究方法、研究結果、批判性思考 |
| NDLTD 連結 | 必須附上（如有） |
| wikilink | >= 2 個概念連結 |

### 每日推薦報告

| 指標 | 最低要求 |
|------|---------|
| 分流完整性 | 每篇論文都必須有判決標籤 |
| 點評品質 | 每篇至少 2 句具體評語（非泛泛之詞） |
| 台灣論文 | 必須標註 🇹🇼 並附學校資訊 |
| 一致性 | 分流表和詳細點評的論文清單必須完全一致 |

## 品質檢查流程

```
筆記生成完成
    │
    ├── 檢查行數 → 不足 → 刪除重來（最多 3 次）
    ├── 檢查公式 → 缺少 → 補充公式
    ├── 檢查圖片 → 缺少 → 嘗試 fallback 取圖
    ├── 檢查必含章節 → 缺少 → 補充章節
    ├── 檢查 wikilink → 不足 → 補充概念連結
    └── 全部通過 → ✅ 儲存筆記
```

## 圖片取得優先順序

1. arXiv HTML 版（`ar5iv.labs.arxiv.org`）直接提取
2. 論文專案主頁
3. PDF 萃取（需 `PyMuPDF`）
4. 無法取得時，用文字描述替代

## 禁止行為

1. **禁止手寫簡化版**：不得用幾句話替代完整模板
2. **禁止跳過論文**：「必讀」論文一篇不能少
3. **禁止編造內容**：所有陳述必須基於論文原文
4. **禁止空白占位**：寧可寫「本文未提供」也不要留空白模板欄位
5. **禁止忽略台灣論文**：台灣論文必須與國際論文同等對待
