## 快速開始

在 Claude Code 中直接輸入以下任一句話，即可啟動對應模式：

```
今日論文推薦                    → 每日論文追蹤（Mode A）
研究 自適應學習                  → 主題深度研究，產出完整論文（Mode B）
讀一下這篇論文 https://arxiv.org/abs/2403.xxxxx  → 單篇精讀（Mode C）
搜尋台灣論文 教育科技             → 台灣博碩士論文專搜（Mode T）
```

就這麼簡單。不需要記指令，用自然語言說就好。

## 四大模式詳細操作

### Mode A：每日論文追蹤

**適用情境**：每天花 5 分鐘掌握領域最新動態。

#### 基本用法

```
今日論文推薦
```

#### 指定天數

```
過去 3 天論文推薦
過去一週的 AI 論文
```

#### 指定領域

```
今日 NLP 論文推薦
今日電腦視覺論文
今日教育科技論文
```

#### 執行過程（自動三步）

```
Step 1 → 抓取論文（arXiv + HuggingFace + 台灣 NDLTD）    ⏱ ~30秒，零 token
Step 2 → AI 點評分流（🔥必讀 / 👀值得看 / ⏭️可跳過）      ⏱ ~2分鐘
Step 3 → 為「必讀」論文生成結構化筆記                       ⏱ 每篇~3分鐘
```

#### 產出範例

```
📁 ~/ObsidianVault/DailyPapers/2026-03-31-論文推薦.md     ← 點評報告
📁 ~/ObsidianVault/論文筆記/Attention Is All You Need.md   ← 結構化筆記
📁 ~/ObsidianVault/論文筆記/_概念/11-深度學習基礎/Transformer.md ← 概念筆記
```

### Mode B：主題深度研究（從想法到論文）

**適用情境**：給一個研究想法，自動跑完文獻回顧→假說→實驗→寫論文。
**預設產出**：APA 第七版格式，15,000-25,000 字，含中英文引用，參考文獻 30+ 篇。

#### 基本用法（預設 APA 格式）

```
研究 生成式AI在偏鄉教育的應用
```

```
寫一篇關於 自適應學習系統對學習成效影響 的論文
```

```
幫我做 AI輔助差異化教學 的文獻回顧
```

#### 指定格式與選項

```
研究 教育科技                        → 預設 APA 格式（15,000-25,000字）
研究 教育科技 --格式 apa_journal     → APA 期刊投稿版（8,000-12,000字）
研究 深度學習 --格式 neurips          → CS 會議格式（5,000-6,500字）
研究 學習分析 --只做到文獻回顧         → 不跑實驗和寫論文
研究 教育科技 --包含台灣論文           → 強調台灣文獻（預設已包含）
```

#### 論文格式比較

| 格式 | 字數 | 引用風格 | 結構 | 適用 |
|------|------|---------|------|------|
| **`apa`**（預設）| **15,000-25,000** | （王大明，2024）/ (Smith, 2024) | 五章式 | 碩博論文 |
| `apa_journal` | 8,000-12,000 | 同上 | Introduction-Method-Results-Discussion | 期刊投稿 |
| `neurips` | 5,000-6,500 | [1][2][3] | Abstract-Intro-Method-Experiments | CS 頂會 |

#### 23 步驟管線流程

```
你只需要說一句話，系統自動執行：

Phase A  研究定義  ──→ 把你的想法轉成明確研究問題
Phase B  文獻探索  ──→ 搜尋國際論文 + 台灣博碩士論文
           ⛩️ 閘門：「這些文獻OK嗎？」（你可以說 OK 或要求重找）
Phase C  知識合成  ──→ 主題聚類 + 生成研究假說
Phase D  實驗設計  ──→ 設計實驗方案 + 自動生成程式碼
           ⛩️ 閘門：「實驗設計OK嗎？」
Phase E  實驗執行  ──→ 在沙盒中跑實驗（自動修復錯誤，最多10輪）
Phase F  分析決策  ──→ 分析結果，決定繼續/調整/換方向
Phase G  論文撰寫  ──→ 大綱→APA格式草稿(15,000-25,000字)→模擬審查→修訂
Phase H  定稿匯出  ──→ 品質閘門→APA引用驗證→匯出 LaTeX/Markdown/Word
           ⛩️ 閘門：「最終品質OK嗎？」
```

#### 中途互動

系統會在三個「閘門」暫停等你確認：
- 你說 **「OK」「通過」「繼續」** → 進入下一階段
- 你說 **「不行，重來」** → 回到前一步重做
- 你說 **「調整方向」** → 換一個研究假說重跑

#### 產出成品

```
📁 artifacts/run-20260331/
├── paper_draft.md         ← APA 格式完整論文（15,000-25,000 字）
├── paper.tex              ← LaTeX 版本（可上傳 Overleaf）
├── references.bib         ← APA 參考文獻（中文在前、英文在後，30+篇）
├── reviews.md             ← 模擬審查意見
├── charts/                ← 自動生成的圖表
├── experiment.py          ← 實驗程式碼
└── verification_report.json ← 引用驗證報告（含 Handle/DOI 驗證）
```

#### APA 格式論文結構（預設）

```
論文產出結構：

摘要 ──────────── 中英文各一份（300-500 字）
第一章 緒論 ────── 研究背景/動機/目的/問題/名詞釋義（2,500+ 字）
第二章 文獻探討 ── 理論基礎/相關研究（含台灣論文）/小結（4,000+ 字）
第三章 研究方法 ── 設計/對象/工具/程序/資料分析（3,000+ 字）
第四章 結果與討論 ─ 描述性統計/推論統計/綜合討論（3,500+ 字）
第五章 結論與建議 ─ 結論/建議/未來方向（1,500+ 字）
參考文獻 ──────── 30+ 篇，中文（筆劃排序）→ 英文（字母排序）
附錄 ──────────── 問卷/量表/程式碼（如有）
```

#### APA 引用範例（論文中的實際樣貌）

```
文內引用：
  中文 → 王大明（2024）指出自適應學習能有效提升學習成效（王大明，2024）。
  英文 → Smith et al. (2024) found that... (Smith et al., 2024)
  混合 → 近年研究（王大明，2024；Chen et al., 2023；Smith, 2024）顯示…

台灣博碩士論文引用：
  王大明（2024）。《AI 輔助差異化教學之研究》〔碩士論文，國立臺灣大學〕。
    臺灣博碩士論文知識加值系統。https://hdl.handle.net/11296/xxxxx

參考文獻排列：
  ┌── 中文文獻（依姓氏筆劃排序）──┐
  │ 王大明（2024）。……              │
  │ 李小華（2023）。……              │
  ├── 英文文獻（依姓氏字母排序）──┤
  │ Chen, Y. (2023). ……            │
  │ Smith, J., & Jones, K. (2024). │
  └──────────────────────────────┘
```

#### 加上篩選條件

```
搜尋台灣論文 自適應學習 臺灣大學

```

#### 直接用 Python 腳本（進階）

```bash
# 基本搜尋
python scripts/taiwan_academic_search.py --keyword "教育科技"

# 指定學校 + 學位
python scripts/taiwan_academic_search.py --keyword "深度學習" --school "臺灣大學" --degree 博士

# 指定年份範圍 + 輸出 JSON
python scripts/taiwan_academic_search.py --keyword "AI" --year-range 2020-2025 --output results.json

# 指定來源
python scripts/taiwan_academic_search.py --keyword "教育" --source ndltd oai --top-n 20
```

## 首次設定

### Step 1：確認環境

```bash
python --version    # 需要 3.11+
pip install pyyaml rich arxiv numpy httpx    # 安裝依賴
```

### Step 2：建立個人配置

```bash
cd D:\AI教學與專案實作\書籍_SKILL\resources\skills\auto-research-claw\config
copy user-config.json user-config.local.json
```

### Step 3：編輯 `user-config.local.json`

修改以下重點欄位：

```jsonc
{
  "paths": {
    "obsidian_vault": "~/ObsidianVault"    // ← 改成你的 Obsidian 路徑
  },
  "daily_papers": {
    "keywords": ["教育科技", "AI education", "adaptive learning"],  // ← 你的研究關鍵字
    "arxiv_categories": ["cs.AI", "cs.CL", "cs.CY"]               // ← 你關注的 arXiv 類別
  },
  "taiwan": {
    "keywords_zh": ["教育科技", "自適應學習", "學習分析"],  // ← 台灣論文搜尋關鍵字
    "preferred_schools": ["臺灣師範大學", "臺灣大學"]       // ← 偏好的學校
  },
  "llm": {
    "model": "deepseek-chat",                               // ← 你用的 LLM
    "base_url": "https://api.deepseek.com"                   // ← API 端點
  }
}
```

### Step 4：設定 API Key（深度研究模式需要）

```bash
# Windows PowerShell
$env:OPENAI_API_KEY = "sk-..."

# 或在系統環境變數中永久設定
```

### Step 5：建立 Obsidian 資料夾（如使用 Obsidian）

```bash
mkdir -p ~/ObsidianVault/DailyPapers
mkdir -p ~/ObsidianVault/論文筆記/_概念/0-待分類
mkdir -p ~/ObsidianVault/論文筆記/_待整理
```

設定完成！直接在 Claude Code 說「今日論文推薦」就能開始了。

## 進階用法

### 組合模式

```
搜尋台灣論文 自適應學習，然後幫我讀前 3 篇
→ 先執行 Mode T 搜尋，再對前 3 篇執行 Mode C 精讀
```

```
今日論文推薦，重點關注教育相關
→ 執行 Mode A，在點評時加重教育相關論文的權重
```

```
研究 AI在偏鄉教育的應用，文獻要包含台灣的碩博士論文
→ 執行 Mode B，Phase B 文獻搜尋自動加入 NDLTD
```

### 只跑部分流程（Mode B）

```
幫我做 學習分析 的文獻回顧
→ 只執行 Phase A-C（研究定義→文獻探索→知識合成），不跑實驗和寫論文

幫我設計 自適應學習 的實驗
→ 執行到 Phase D（實驗設計），不實際跑實驗
```

### 切換論文格式

```
研究 教育科技                       → APA 完整版（預設，15,000-25,000字）
研究 教育科技 --格式 apa_journal    → APA 期刊版（8,000-12,000字）
研究 深度學習 --格式 neurips         → NeurIPS 會議格式（5,000-6,500字）
研究 自然語言處理 --格式 icml        → ICML 會議格式（5,000-6,500字）
```

也可以在 `user-config.local.json` 中永久設定預設格式：
```json
{ "deep_research": { "paper_format": "apa" } }
```

### 批次讀論文

```
幫我讀以下論文：
1. https://arxiv.org/abs/2403.11111
2. https://arxiv.org/abs/2403.22222
3. https://ndltd.ncl.edu.tw/r/abc123
→ 逐篇在 fork 中並行處理
```

### 匯出格式

Mode B 產出的論文支援：
- **Markdown**（`.md`）→ 在 Obsidian 中閱讀，APA 格式完整保留
- **LaTeX**（`.tex` + `.bib`）→ 直接上傳 Overleaf
- **PDF**（需安裝 LaTeX 編譯器）
- **Word**（`.docx`）→ 搭配 `/docx` 技能轉檔

## 常見問題

### Q: 需要付費嗎？
論文抓取（Mode A Step 1）和台灣論文搜尋（Mode T）**完全免費**，零 token 消耗。
AI 點評、筆記生成、深度研究需要 LLM API（如 DeepSeek 每天約 0.2 元人民幣）。

### Q: 沒有 Obsidian 可以用嗎？
可以。筆記會生成為標準 Markdown 檔案，用任何編輯器都能開。
只是 `[[wikilink]]` 和 MOC 功能在 Obsidian 中體驗最好。

### Q: 台灣論文搜不到怎麼辦？
1. 嘗試不同的中文同義詞（如「人工智慧」vs「AI」vs「機器學習」）
2. 縮小年份範圍（NDLTD 開放資料只到 2023 年）
3. 系統會自動提供 NDLTD 網頁搜尋連結，可以在瀏覽器中手動搜尋

### Q: 論文能直接投稿嗎？
Mode B 產出的論文是高品質草稿，但仍需要人工審閱和修改。
系統會模擬同儕審查並指出問題，但最終投稿前請務必自行檢查。

### Q: 支援哪些 LLM？
任何 OpenAI 相容的 API 都可以：
- DeepSeek（推薦，便宜好用）
- GPT-4o / GPT-5.4
- Gemini（透過相容 API）
- 本地模型（Ollama 等）

### Q: 引用格式可以改嗎？
**預設就是 APA 第七版**（含中英文混合引用）。如需切換：
- `"paper_format": "apa"` → APA 完整版，15,000-25,000 字（碩博論文，預設）
- `"paper_format": "apa_journal"` → APA 期刊版，8,000-12,000 字
- `"paper_format": "neurips"` → NeurIPS 數字引用，5,000-6,500 字

### Q: APA 引用格式正確嗎？
系統內建完整的 APA 第七版規範，包含：
- 英文引用：(Smith, 2024) / Smith et al. (2024)
- 中文引用：（王大明，2024）/ 王大明等人（2024）
- 台灣博碩士論文：王大明（2024）。《標題》〔碩士論文，國立臺灣大學〕。臺灣博碩士論文知識加值系統。
- 參考文獻排列：中文筆劃排序在前 → 英文字母排序在後
- 但最終投稿前仍建議人工檢查一遍格式細節

### Q: 論文字數夠多嗎？
APA 完整版產出 **15,000-25,000 字**（不含參考文獻），各章節有最低字數要求：
- 緒論 2,500+ 字 / 文獻探討 4,000+ 字 / 研究方法 3,000+ 字
- 結果與討論 3,500+ 字 / 結論與建議 1,500+ 字
- 參考文獻 30+ 篇

**問題：Python 版本不相容**
本 SKILL 需要 Python 3.11+。檢查版本：
```bash
python --version
```

### API 連線問題

**問題：arXiv API 超時**
- 系統會自動重試最多 2 次（間隔 5 秒）
- 如持續失敗，檢查網路連線或稍後再試
- arXiv API 有速率限制，避免短時間內大量請求

**問題：HuggingFace API 回傳空結果**
- 某些日期可能沒有 Daily Papers（週末較少）
- 系統會自動重試，如仍為空屬正常現象

**問題：NDLTD 被 CAPTCHA 阻擋**
- NDLTD Web Search 可能會觸發驗證碼
- 系統會自動退回到 Handle API 取基本資訊
- 也可改用 `--source ndltd`（OpenData CSV）代替網頁搜尋

### 路徑問題

**問題：找不到輸出檔案**
- Windows 路徑：`%USERPROFILE%\tmp\daily_papers_top30.json`
- Linux/Mac 路徑：`/tmp/daily_papers_top30.json`
- 自訂路徑：`python fetch_papers.py --output /your/path/output.json`

**問題：Obsidian Vault 路徑錯誤**
編輯 `config/user-config.local.json`：
```json
{
  "paths": {
    "obsidian_vault": "D:/你的實際路徑/ObsidianVault"
  }
}
```

### 台灣論文搜尋問題

**問題：OAI-PMH 機構典藏搜不到結果**
- 各校 OAI-PMH 端點可能暫時維護
- 嘗試指定不同學校：`--school 臺灣大學`
- 某些學校限制每次請求的記錄數，系統會自動分頁（最多 3 頁）

**問題：Crossref 搜尋結果不相關**
- Crossref 使用英文標題搜尋，建議同時提供英文關鍵字
- 台灣 DOI prefix `10.6342` 可幫助篩選台灣來源
