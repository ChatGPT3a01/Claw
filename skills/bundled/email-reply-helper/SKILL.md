# Email 回覆助手

## 用途

當使用者提供一封客戶來信，本技能會：

1. 分析來信的情緒和主要需求
2. 判斷回覆類型（詢價/客訴/合作/一般）
3. 根據公司規範產生專業回覆信件

## 前置檢查

在開始執行前，MUST 先檢查以下條件：

1. Check if 使用者是否提供了客戶來信原文
   - 若否 → Ask user「請提供客戶來信的完整內容」

2. Check if 來信內容是否為中文
   - 若否 → 詢問使用者「是否需要以中文回覆？」

3. Check if 來信中是否包含收件人資訊
   - 若否 → Ask user「請提供收件人的稱謂（如：王先生）」

## 執行步驟

### Step 1：分析來信

- 閱讀使用者提供的客戶來信
- 識別來信者的情緒（正面/中性/負面）
- 摘要出 3 個重點需求

### Step 2：判斷回覆類型

- Check if 來信內容包含價格詢問 → 分類為「詢價」
- Check if 來信內容包含抱怨或不滿 → 分類為「客訴」
- Check if 來信內容包含合作邀請 → 分類為「合作洽談」
- 以上皆非 → 分類為「一般往來」

### Step 3：組織回覆內容

- MUST 以「○○先生/小姐 您好：」開頭
- MUST 先回應對方的主要需求
- MUST 提供具體的下一步行動
- MUST 以「敬祝 商祺」結尾

### Step 4：最終檢查

- NEVER 使用口語化表達（如：嗨、OK、沒問題啦）
- NEVER 承諾未經確認的價格或交期
- NEVER 洩露公司內部資訊
- Check if 金額超過 10 萬元，Ask user 是否需要加入主管副本說明

## 例外處理

### 遇到以下情況時的處理方式：

1. **來信內容涉及法律爭議**
   - NEVER 直接回覆法律相關承諾
   - MUST 建議使用者「本信件涉及法律事項，建議轉交法務部門處理」

2. **來信包含人身攻擊或不當言語**
   - MUST 保持專業態度回覆
   - NEVER 以攻擊性語言回應
   - 建議使用者考慮是否需要主管介入

3. **來信內容不完整，無法判斷需求**
   - 撰寫「詢問確認信」而非直接回覆
   - 列出需要對方補充的資訊項目

## 品質檢查

回覆信件完成後，MUST 自行檢查以下項目：

| 檢查項目 | 要求                         |
| -------- | ---------------------------- |
| 語氣     | 必須為正式商業用語           |
| 字數     | 回覆內容 100-500 字之間      |
| 段落     | 每段不超過 3 句話            |
| 格式     | 使用項目符號列舉重點         |
| 用詞     | 無錯別字、無口語化表達       |

如果任何一項不通過，MUST 自動修正後再輸出。

## 安全規則

### 絕對不可輸出的內容

- NEVER 包含公司內部售價或成本資訊
- NEVER 提及競爭對手的負面評價
- NEVER 承諾未經授權的折扣或優惠
- NEVER 使用帶有性別歧視或任何歧視性的用語

### 需要人工確認的情況

- 金額超過 50 萬 → Ask user 確認是否需要主管審核
- 涉及退貨或賠償 → Ask user 確認公司政策
- 對方要求立即回覆具體交期 → Ask user 提供確認的交期

## 輸出格式

請依照以下格式輸出：

### 分析摘要

- 來信情緒：[正面/中性/負面]
- 回覆類型：[詢價/客訴/合作/一般]
- 重點需求：
  1. [需求一]
  2. [需求二]
  3. [需求三]

### 回覆信件

---

主旨：Re: [原始主旨]

[收件人稱謂] 您好：

[回覆內容]

[結尾敬語]

[寄件人資訊]

---

### 執行摘要

- 來信分類：[類型]
- 處理方式：[直接回覆/需確認/轉交處理]
- 風險提醒：[無/低/中/高]
- 建議後續：[無需跟進/需在 X 天內追蹤/需主管介入]

## 參考資料

撰寫回覆時，MUST 參考以下範本：

- 信件格式請遵循 `@assets/email-template.md` 中定義的標準
- 依據來信類型選擇對應的開頭和結尾用語

## 範例

### 輸入範例

> 客戶來信：
> 「您好，我們公司想詢問貴司 A 型號產品 500 組的報價，
> 希望能在下個月 15 日前交貨，請問可以嗎？」

### 預期輸出

#### 分析摘要

- 來信情緒：中性
- 回覆類型：詢價
- 重點需求：
  1. A 型號產品報價
  2. 數量 500 組
  3. 交期需求：下個月 15 日前

#### 回覆信件

---

主旨：Re: A 型號產品詢價

○○先生/小姐 您好：

感謝您對本公司 A 型號產品的詢問。

關於您提到的需求，回覆如下：

1. **A 型號產品 500 組報價**，我們將於 1 個工作天內提供正式報價單
2. **交期部分**，我們需確認目前庫存狀況後回覆，預計明日告知

如有其他問題，歡迎隨時聯繫。

敬祝 商祺

[你的姓名]
[職稱] | [公司名稱]
[聯絡電話] | [Email]

---

#### 執行摘要

- 來信分類：詢價
- 處理方式：直接回覆
- 風險提醒：低（需確認交期可行性）
- 建議後續：需在 1 個工作天內提供報價單
