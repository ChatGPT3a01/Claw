# 公司標準 Email 範本

## 信件開頭（選擇適合的）

- 一般往來：「○○先生/小姐 您好：」
- VIP 客戶：「○○總經理/董事長 您好，感謝您的來信：」
- 初次聯繫：「○○先生/小姐 您好，初次聯繫，冒昧打擾：」

## 常用結尾

- 一般：「如有任何問題，歡迎隨時聯繫。敬祝 商祺」
- 需後續追蹤：「我們將於 [X] 個工作天內提供完整回覆。敬祝 商祺」
- 緊急回覆：「已立即處理，以下是我們的回應。敬祝 順心」

## 公司簽名檔

```
[姓名]
[職稱] | [部門]
[公司名稱]
Tel: [電話] | Email: [信箱]
---
本信件僅供收件人使用，如非指定收件人請刪除。
```

## 語氣對照表

| 來信類型 | 建議語氣             | 開頭用語                     |
| -------- | -------------------- | ---------------------------- |
| 詢價     | 積極、專業           | 感謝您對本公司產品的詢問     |
| 客訴     | 誠懇、負責           | 非常感謝您的反饋，對此深表歉意 |
| 合作洽談 | 友善、開放           | 感謝您的合作提議，我們非常有興趣 |
| 一般往來 | 禮貌、簡潔           | 感謝您的來信                 |

### 金額格式

- MUST 使用千分位格式：NT$ 1,000,000
- MUST 加上幣別前綴：NT$ / USD$
- 範圍表示：NT$ 500,000 ~ NT$ 800,000
