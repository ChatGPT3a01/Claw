# 設計風格參考指南

## 常用色彩方案

### 現代科技風
- 主色：#667eea → #764ba2（紫色漸層）
- 背景：#f8f9fa（淺灰）
- 文字：#1a1a2e（深色）
- 強調：#00d2ff（亮藍）

### 企業專業風
- 主色：#1e3c72 → #2a5298（深藍漸層）
- 背景：#ffffff（白色）
- 文字：#333333（深灰）
- 強調：#28a745（綠色）

### 活潑創意風
- 主色：#f093fb → #f5576c（粉紫漸層）
- 背景：#fafafa（極淺灰）
- 文字：#2d3436（炭黑）
- 強調：#fdcb6e（暖黃）
