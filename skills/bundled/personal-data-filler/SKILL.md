## 支援的檔案格式

### 1. **Word 文件 (.docx)**
- 自動填寫標記欄位（如 `[姓名]`、`{{phone}}`）
- 填寫表單欄位
- 替換模板中的預留位置

### 2. **Excel 試算表 (.xlsx)**
- 填寫指定儲存格（如 A1:A6）
- 根據列標頭自動匹配欄位
- 支援多個工作表

### 3. **PDF 表單 (.pdf)**
- 填寫 PDF 表單欄位
- 支援簽名欄位
- 生成填寫後的 PDF

### 4. **HTML/Web 表單**
- 識別並填寫表單欄位
- 支援各種輸入類型（text、email、tel、hidden）
- 生成填好的 HTML 或已提交狀態

### 5. **JSON 檔案**
- 自動填寫結構化資料
- 支援嵌套物件和陣列

### 6. **純文字檔案**
- 尋找並替換預設字段如：`{姓名}`、`${phone}`、`[address]`

## 使用方式

### 基本用法

提供任何包含個人資訊欄位的檔案，我會自動填寫：

```
使用者：我有一份表單需要填寫個人資料。
我：我會自動填入你的個人資訊。請提供檔案。
```

### 自訂模板符號

如果檔案中使用特定的預留位置標記，我會自動識別：

- `[欄位名]` - 方括號
- `{欄位名}` - 花括號
- `{{欄位名}}` - 雙花括號
- `${欄位名}` - 美元符號加花括號
- `__欄位名__` - 雙底線

### 批量填寫

提供多個檔案，我可以逐個填寫所有檔案：

```
使用者：幫我填寫這 5 個表單。
我：我會依次填寫每個檔案並提供結果。
```

## 工作流程

1. **接收檔案** - 使用者提供需要填寫的檔案
2. **分析格式** - 我識別檔案類型和欄位結構
3. **自動填寫** - 用個人資料填寫所有相關欄位
4. **驗證** - 確認所有資訊正確填入
5. **輸出** - 提供已填寫的檔案

## 常見問題

**Q：如果檔案中的欄位名稱與我的資訊不完全匹配怎麼辦？**
A：我會嘗試進行智能匹配（例如「聯絡電話」→「電話」），如果無法確定則會詢問。

**Q：可以只填寫部分欄位嗎？**
A：可以。指定要填寫的欄位，我會只更新指定欄位。

**Q：填寫後的檔案會覆蓋原檔案嗎？**
A：不會。我會產生新的檔案副本，原檔案保持不變。

**Q：支援其他檔案格式嗎？**
A：主要支援上述格式。如需其他格式，請告知。
