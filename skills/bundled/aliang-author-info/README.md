# 阿亮老師作者資訊模板 Skill

## 📝 簡介

這是一個自定義 Claude Code Skill，用於快速在任何 GitHub 專案中套用阿亮老師的作者資訊、授權聲明和社群連結。

## 🚀 使用方式

在 Claude Code 對話中，使用以下任一指令：

```
/aliang-author-info
```

或直接說：
- "套用作者資訊"
- "加入作者模板"
- "使用阿亮老師模板"

## ✨ 功能特點

- ✅ 自動複製作者照片到專案目錄
- ✅ 美化的 Markdown 格式（居中對齊、表格排版）
- ✅ 專業的社交媒體徽章按鈕
- ✅ 完整的授權聲明
- ✅ 精美的頁尾區塊
- ✅ 自動 Git 提交（可選）

## 📂 包含的資訊

### 作者資訊
- 姓名、職務、獲獎紀錄
- 專業的表格排版

### 社群連結
- YouTube 頻道
- Facebook 個人頁面
- 3A科技研究社
- Email 聯絡方式

### 授權聲明
- 版權聲明
- 使用限制
- 禁止事項

## 📸 效果預覽

套用後的 README 會包含：

```markdown
## 👨‍🏫 關於作者
[美化的作者資訊區塊]

## 📜 授權聲明
[完整的授權說明]

---
[精美的頁尾]
```

## 🔧 自定義

如需修改模板內容，編輯：
```
D:\提示詞模板\.claude\skills\aliang-author-info\instructions.md
```

> 💡 **Skill 位置**：所有 Skill 統一放在 `D:\提示詞模板\.claude\skills\` 目錄下

## 📝 注意事項

1. 確保專案目錄有 README.md 檔案
2. 作者照片會自動從 `D:\提示詞模板\作者資訊.png` 複製
3. 年份會根據當前時間自動更新
4. 可選擇是否自動 Git commit
