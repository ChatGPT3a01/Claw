## 專案簡介

**亮言~NotebookLM 自動化Skill** 是一個 Flask Web 應用程式，讓你透過網頁介面，用自然語言控制 Google NotebookLM。

### 🎯 核心功能

- 📚 筆記本管理（建立、刪除、重命名）
- 📎 來源匯入（URL、YouTube、PDF）
- 🎙️ 內容生成（Podcast、影片、簡報、測驗、閃卡、心智圖等）
- 🔍 智慧問答（對文件提問）
- 📥 下載匯出

### 🏗️ 技術架構

```
┌─────────────────────────────────────┐
│     Flask Web GUI (前端介面)         │
├─────────────────────────────────────┤
│     NLP Parser (自然語言解析)        │
│   ┌─────────┬─────────┬─────────┐   │
│   │ 關鍵字  │ Gemini  │ OpenAI  │   │
│   └─────────┴─────────┴─────────┘   │
├─────────────────────────────────────┤
│     notebooklm-py (CLI 封裝)         │
├─────────────────────────────────────┤
│     Playwright (瀏覽器自動化)        │
├─────────────────────────────────────┤
│     NotebookLM 網頁                  │
└─────────────────────────────────────┘
```

### 核心模組說明

#### 1. NLP Parser（自然語言解析器）

**檔案：** `services/nlp_parser.py`

**功能：** 將使用者輸入的自然語言轉換為系統可理解的意圖（intent）

**實作方式：**

```python
# 意圖模式 - 關鍵字匹配
INTENT_PATTERNS = {
    "list_notebooks": [
        "列出筆記本", "顯示筆記本", "我的筆記本"
    ],
    "create_notebook": [
        "建立筆記本", "新增筆記本", "創建筆記本"
    ],
    "generate_audio": [
        "生成Podcast", "製作音訊", "產生播客"
    ],
    # ... 更多意圖
}
```

**解析流程：**
```
使用者輸入：「幫我生成 Podcast」
     ↓
關鍵字匹配：找到 "生成Podcast" 或 "Podcast"
     ↓
回傳意圖：{ "intent": "generate_audio", "confidence": 0.9 }
```

#### 2. NotebookLM Service（CLI 封裝）

**檔案：** `services/notebooklm_service.py`

**功能：** 封裝 notebooklm CLI 指令，提供 Python 介面

**實作方式：**

```python
def _run_cli(self, args: List[str], timeout: int = 120):
    """執行 notebooklm CLI 指令"""
    cmd = ["notebooklm"] + args
    result = subprocess.run(
        cmd,
        capture_output=True,
        text=True,
        timeout=timeout
    )
    # 解析 JSON 回傳
    return {"success": True, "data": json.loads(result.stdout)}

def list_notebooks(self):
    return self._run_cli(["list", "--json"])

def generate_audio(self, notebook_id=None):
    args = ["generate", "audio", "--json"]
    if notebook_id:
        args.extend(["--notebook", notebook_id])
    return self._run_cli(args, timeout=60)
```

#### 3. Execute API（執行端點）

**檔案：** `routes/execute.py`

**功能：** 接收自然語言，解析後執行對應操作

**實作方式：**

```python
@api_bp.route('/execute', methods=['POST'])
def execute_command():
    command = request.json.get('command')

    # 1. 解析自然語言
    parsed = nlp_parser.parse(command)
    intent = parsed.get('intent')

    # 2. 根據意圖執行
    if intent == "list_notebooks":
        return notebooklm_service.list_notebooks()
    elif intent == "generate_audio":
        return notebooklm_service.generate_audio()
    # ... 更多意圖處理
```

#### 4. 背景任務管理

**檔案：** `services/task_manager.py`

**功能：** 處理長時間任務（如 Podcast 生成），避免前端等待逾時

**實作方式：**

```python
class TaskManager:
    def __init__(self):
        self.tasks = {}

    def create_task(self, name, func, kwargs=None):
        task_id = str(uuid.uuid4())[:8]
        thread = threading.Thread(
            target=self._run_task,
            args=(task_id, func, kwargs)
        )
        thread.start()
        return task_id

    def get_task(self, task_id):
        return self.tasks.get(task_id)
```

## 部署方式

### 方式一：本機使用（最簡單）

```bash
cd D:\NotebookLM課程\讓NotebookLM自動化
python app.py
```

訪問 http://localhost:5000

### 方式二：區域網路分享

修改 `app.py`：

```python
if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000, debug=True)
```

其他電腦訪問：`http://你的IP:5000`

> ⚠️ 注意：需要先在伺服器電腦執行 `notebooklm login`

### 方式三：使用 Waitress（Windows 正式部署）

```bash
pip install waitress
```

建立 `run_server.py`：

```python
from waitress import serve
from app import app

if __name__ == '__main__':
    print("伺服器啟動於 http://0.0.0.0:5000")
    serve(app, host='0.0.0.0', port=5000)
```

執行：
```bash
python run_server.py
```

### 方式四：使用 Gunicorn（Linux/macOS 正式部署）

```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

### 方式五：Docker 部署

建立 `Dockerfile`：

```dockerfile
FROM python:3.11-slim

WORKDIR /app

# 安裝依賴
RUN pip install notebooklm-py flask
RUN pip install "notebooklm-py[browser]"
RUN playwright install chromium
RUN playwright install-deps

# 複製專案
COPY . .

# 啟動
EXPOSE 5000
CMD ["python", "app.py"]
```

建立並執行：
```bash
docker build -t notebooklm-gui .
docker run -p 5000:5000 -v ~/.notebooklm:/root/.notebooklm notebooklm-gui
```

> ⚠️ 需要掛載登入狀態目錄

### 方式六：開機自動啟動（Windows）

1. 建立 `start.bat`：
```batch
@echo off
cd /d D:\NotebookLM課程\讓NotebookLM自動化
python app.py
pause
```

2. 按 `Win+R`，輸入 `shell:startup`
3. 將 `start.bat` 捷徑放入啟動資料夾

## 常見問題

### Q1: 出現「No notebook specified」錯誤

**原因：** 未選擇筆記本就執行需要筆記本的操作

**解決：** 先在左側下拉選單選擇筆記本，或先執行「列出筆記本」再選擇

---

### Q2: 登入狀態失效

**原因：** Cookie 過期或被清除

**解決：** 重新執行 `notebooklm login`

---

### Q3: 生成 Podcast 很久沒反應

**原因：** Podcast 生成需要時間（約 2-5 分鐘）

**解決：**
- 檢查右側「任務狀態」欄位
- 任務會在背景執行，完成後會顯示

---

### Q4: NLP 解析不準確

**原因：** 關鍵字匹配模式有限制

**解決：**
1. 使用更明確的關鍵字（如「生成 Podcast」而非「做個音檔」）
2. 在設定頁切換為 Gemini 或 OpenAI 模式

---

### Q5: 無法遠端存取

**原因：** 防火牆阻擋或 host 設定問題

**解決：**
1. 確認 `app.run(host='0.0.0.0')`
2. 開放防火牆 5000 port
3. Windows: `netsh advfirewall firewall add rule name="Flask" dir=in action=allow protocol=tcp localport=5000`

---

### Q6: 如何更新 notebooklm-py

```bash
pip install --upgrade notebooklm-py
```
