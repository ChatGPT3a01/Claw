# 提案審查器

## 審查項目
| 面向 | 檢查重點 |
|------|----------|
| 完整性 | 是否涵蓋所有核心需求？ |
| 說服力 | 是否有明確的價值主張？ |
| 可行性 | 時程和預算是否合理？ |
| 格式 | 是否符合專業商務文件格式？ |

## 執行步驟
- 逐項檢查上表的四個面向
- MUST 列出發現的所有問題（按嚴重程度排序）
- MUST 提供每個問題的具體修改建議
- 自動修正明確的錯誤（錯字、格式不一致）
- Ask user 確認需要人工判斷的重大修改
- NEVER 降低提案書的專業水準
